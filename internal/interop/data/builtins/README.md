# Built-in stdlib overrides

`.esc` override files for ECMAScript standard-library classes. One file
per ECMAScript spec revision (`es5.esc`, `es2015.esc`, …) plus
`dom.esc`. See the "Authoring policy for §6.B/§6.C" subsection of
§6 Phasing in `planning/interop_mutability/implementation_plan.md`
for the per-class checklist and authoring policy.

This directory is empty at the §6.A infrastructure phase; §6.B and §6.C
populate it.
