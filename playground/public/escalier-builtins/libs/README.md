# Third-party library overrides

`.esc` override files for FP / immutability libraries (Ramda, fp-ts,
Effect, Immutable.js, lodash/fp). One file per library, typically
carrying a module-level `@all_pure` pragma. See §6 Phasing of
`planning/interop_mutability/implementation_plan.md`.

This directory is empty at the §6.A infrastructure phase; §6.E
populates it.
